# FA Closer Hub

Sistema corporativo interno da Fazendo Acontecer (FA) para geração otimizada de relatórios comerciais por Closers, focado em agilidade (Zero Latency), métricas precisas (Oportunidades Novas e Cadência) e exportação em PDF corporativo.

## Tecnologias Usadas
- Next.js (App Router)
- React 19 + TypeScript
- Tailwind CSS v4
- jsPDF (geração de relatórios client-side)
- Framer Motion (micro-interações premium)

## Visual Identity (Aura Style)
Design estritamente corporativo FA:
- Dark mode severo (fundo Absolute Black ~#050505)
- Contrastes elegantes com Branco/Cinza e Glassmorphism 
- Grid animado background super sutil focado na retenção de atenção
- Minimalismo de inputs com validações eficientes

## Como Rodar Localmente
```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
